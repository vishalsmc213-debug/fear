# FEAR - Future Education & AI Resources 🚀

**FEAR** என்பது மாணவர்களுக்கான ஒரு அட்வான்ஸ்டு AI தளம். இதில் புராஜெக்ட் ஐடியாக்கள், சர்க்யூட் டயக்ராம்கள் மற்றும் AI மூலமாக பிழைகளைத் திருத்தும் வசதிகள் உள்ளன.

## 🛠 அம்சங்கள் (Features)
- **Smart Project AI:** ₹10-20 விலையில் தனித்துவமான புராஜெக்ட் ஐடியாக்கள்.
- **AI Voice Ask:** குரல் வழியாக கேள்விகளைக் கேட்கலாம் (Gemini AI).
- **Code Error Fixer:** கோடிங்கில் உள்ள பிழைகளை ChatGPT மூலம் சரிசெய்யலாம்.
- **Subscription Model:** 1 வாரம் இலவசம், பின் ₹50/மாதம்.

## 🚀 தொழில்நுட்பங்கள் (Tech Stack)
- **Frontend:** HTML5, CSS3 (Neon UI), JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI Models:** OpenAI (ChatGPT), Google Generative AI (Gemini 1.5 Flash)

## 📦 நிறுவும் முறை (Installation)
1. ரிப்போசிட்டரியை குளோன் செய்யவும்: `git clone https://github.com/yourusername/fear-ai.git`
2. சர்வர் ஃபோல்டருக்குச் செல்லவும்: `cd server`
3. தேவையானவற்றை இன்ஸ்டால் செய்யவும்: `npm install`
4. உங்கள் API கீகளை `.env` கோப்பில் சேர்க்கவும்.
5. சர்வரை இயக்கவும்: `node server.js`
